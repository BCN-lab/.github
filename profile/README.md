# Behavioral Cognitive Neuroscience Lab

Code, tools, documentation, and resources from the **Behavioral Cognitive Neuroscience (BCN) Lab** led by [Dr. Matthias Nau](https://matthiasnau.com/). 

## What you can find here
This GitHub organization contains repositories for:

* **Research code** associated with lab projects and publications
* **Reusable tools and utilities** for behavioral and neuroimaging studies
* **Documentation and manuals** for lab members and collaborators
* **Teaching and training materials** related to coding, analysis, and open science

## OpenMReye

A major open-source project from the lab is **OpenMReye**, a framework for MR-based eye tracking without a camera for research and clinical applications.

* [OpenMReye website](https://matthiasnau.com/openmreye/)
* [OpenMReye GitHub organization](https://github.com/deepmreye)
